# **AutoMathDB 개발 우선순위 로드맵 (MVP 기반)**

**목표:** 텍스트/수식 추출부터 클라우드(Firebase) 업로드까지, 가장 핵심적인 '데이터 파이프라인'을 관통시키는 뼈대 완성.

## **🏁 Phase 0: 사전 준비 및 기획 (Pre-requisites)**

* \[x\] **입력 데이터 표준화:** 업로드할 PDF의 포맷을 \[문제 \- 정답/해설 \- 접근법/개념/풀이단계\]가 페이지별로 분리된 형태로 고정. (AI 정확도 극대화)  
* \[x\] **MVP 스펙 확정:** 첫 번째 버전에서는 '이미지 추출'을 제외하고 텍스트와 수식(LaTeX)을 100% 완벽하게 추출하는 데 집중하기로 결정.  
* \[x\] **클라우드 인프라 준비:** 관리자(태경님)가 직접 구글 Firebase 콘솔에 접속하여 새 프로젝트 및 데이터베이스(Firestore) 생성 완료.

## **🟢 Phase 1: 기본 UI 및 프론트엔드 뼈대 (완료)**

* \[x\] **단일 환경 구축:** index.html 뼈대 생성.  
* \[x\] **디자인 및 라이브러리 연동:** Tailwind CSS, FontAwesome(아이콘), PDF.js 연결.  
* \[x\] **UI 컴포넌트 배치:** 설정, 업로드, 대시보드 표.

## **🟢 Phase 2: PDF 파싱 및 AI 추출 엔진 (완료 \- Vision 방식 전환)**

* \[x\] **PDF 렌더링 (New):** PDF.js를 활용해 각 페이지를 고해상도 이미지(Canvas)로 캡처.  
* \[x\] **Gemini Vision AI 연동 (New):** 추출된 이미지들을 캔버스의 내장 멀티모달 AI에게 전달.  
* \[x\] **✨ 시멘틱 인텐트 프롬프트 강화:** AI에게 시각적 지시 추가 적용. 단순 힌트 대신 **접근법, 개념, 풀이단계** 3가지 항목으로 세분화.  
* \[x\] **🏷️ 개념 필드 최적화 (New):** DB 검색 및 분류 효율을 위해 '개념' 항목의 긴 설명을 무시하고 \*\*핵심 키워드(용어)만 추출(쉼표 구분)\*\*하도록 프롬프트 고도화 완료.  
* **\[🚨 트러블슈팅 및 버그 픽스 내역\]** \* \[x\] CSP Worker 차단 에러: HTML 내부에 일꾼 스크립트 직접 선언.  
  * \[x\] 마크다운(Regex) 파싱 에러: 첫 \[와 끝 \]만 추적하여 100% 순수 데이터만 발췌.  
  * \[x\] alert() 샌드박스 에러: 자체 개발 알림창(Toast UI)으로 교체.  
  * \[x\] 2단 레이아웃 파싱 오류: '이미지 스캔(Vision) 기반 추출'로 업그레이드 완료.  
  * \[x\] **\[긴급 수정\] AI 답변 끊김 해결:** maxOutputTokens를 8192로 늘리고 예외 처리 로직 추가.  
  * \[x\] **PDF.js 패턴 렌더링 오류 해결:** 버전 3.11.174로 업그레이드 및 Worker 설정 추가.

## **🟢 Phase 3: 데이터 검수 및 중복 방지 (완료)**

* \[x\] **대시보드 동적 렌더링:** 성공적으로 뽑아낸 JSON 데이터를 7개 열(문제/정답/해설/접근법/개념/단계)의 표에 뿌려주기.  
* \[x\] **인라인 수정 기능:** 표의 셀을 클릭하면 즉석에서 고칠 수 있도록 속성 부여.  
* \[x\] **디지털 지문(Hash) 암호화:** 추출된 텍스트를 고유한 해시 암호로 변환.

## **🟢 Phase 4: 클라우드 DB 연동 및 내보내기 (완료)**

* \[x\] **Firebase SDK 연동:** 모듈(Module) 스크립트 방식으로 Firebase v11 라이브러리 연동 완료.  
* \[x\] **DB 업로드 및 중복 차단:** signInAnonymously 익명 인증 적용 및 해시(Hash) 값을 문서 고유 ID로 지정하여 중복 덮어쓰기 로직 구현.  
* \[x\] **엑셀 내보내기:** 브라우저 내장 Blob 객체와 \\uFEFF(BOM) 처리를 통해 한글 깨짐이 없는 CSV 파일 다운로드 기능 탑재.  
* \[x\] **\[버그 픽스\] Canvas 임시 DB 가로채기 방지:** Canvas 미리보기 환경의 임시 \_\_firebase\_config 변수 주입을 무시하고, 원장님의 실제 Firebase 설정값을 하드코딩하여 소유권 연동 강제 고정 완료.  
* \[x\] **\[보안 규칙\] 403 Forbidden 에러 해결:** Firestore 보안 규칙(Rules) 게시를 통해 인증된 사용자(익명 로그인)의 데이터 쓰기 권한 부여 완료.  
* \[x\] **대시보드 실시간 현황판 (New):** getCountFromServer를 활용하여 서버 과부하 없이 DB에 누적된 총 문항 수를 실시간으로 렌더링하고, 현재 사용 중인 AI 모델명 표시 기능 구현 완료.
